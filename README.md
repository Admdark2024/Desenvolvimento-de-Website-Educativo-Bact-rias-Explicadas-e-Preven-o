# MicroMundo

Website educativo e interativo sobre bactérias, doenças e prevenção.

## Recursos

- Comparação entre bactérias e vírus
- Cards interativos sobre gonorreia, sífilis, hanseníase e tuberculose
- Orientações práticas de prevenção
- Seção de mitos e verdades
- Quiz com pontuação e certificado meme imprimível
- Layout responsivo e acessível

## Executar localmente

Abra `outputs/index.html` no navegador. O projeto usa somente HTML, CSS e JavaScript, sem etapa de compilação.

## Deploy no Vercel

1. Importe este repositório no Vercel.
2. Selecione **Other** como framework.
3. Mantenha o comando de build vazio.
4. Faça o deploy. O arquivo `vercel.json` encaminha a raiz para o site em `outputs/`.

## Aviso

O conteúdo é educativo e não substitui avaliação ou orientação de profissionais de saúde.
